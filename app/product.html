CONTACT CARD (SCard XCT136)

SCard XCT136 is a member of SCard family device from Xirka Silicon Technology which is designed specifically for contact interface card. SCard is CPU based card. It is ideal for cost sensitive market that requires low cost solution. The chip enables machines to connect to cellular networks using GSM Phase 2 standard. The chip has on-chip crypto accelerator that ensures high security while maintaining processor performance. The CPU also cooperates with secured memory management and secured bus in order to increase data security. Internal Power management ensures low power consumptions that increase devices operation times. The flexibility of hardware/software development tools and optimized architecture for speed and power ensure to meet customer requirements.

    Features

  • ISO 7816-3 compliant
  • Supported class A and B
  • ETU Timer/Counter

    Processor

  • 32 bit microprocessor
  • Optimized for speed and power
  • Internal general lock

    Memory

  • Flash : 136 Kbyte
  • RAM : 4 Kbyte

VeriU XVU 136S256L

VeriU XVU 136S/256L is a Contactless or dual interface Smartcard wich designed specifically for high security samrt card. The VeriU family is a CPU based card using 8/16 Bit microprocessor with optimized architecture for power and speed. For dual interface VeriU supports contact interface based on ISO/IEC 7816 class A/B and contactless interface based on 14443 type standard protocol.

    Interface

  • Support contact interface based ISO 7816 Class A and Class B.
  • Support contactless interface based on ISO/IEC 14443 Type A.
  • NFC Forum Type 4 Compliant Tag IC.

    Features And Security

  • High performance Public Key Infrastructure (PKI)
  • True Random Generator
  • Data integrity using CRC-16 (ISO 14443 type A)
  • Symmetrical encryption: DES, 3DES, AES.
  • Secured bus and memory management
  • On-Chip voltage regulator

    PROCESSOR

  • 8/32 Bit Processor
  • Internal generated clock to allow the device to be independent of the external clock
  • DMA module to improve the performance of data transfer
  • Architecture optimized for speed and power

    MEMORY

  • Flash 136/256 KBPS
  • RAM 6 Kbyte
  • Selectable Code and Data Banking
  • Support OTP master key insertion

    POWER MANAGEMENT

  • IDLE Mode (Low current stnadby information)
  • ACTIVE Mode
  • I/O Transmission with CPU in idle mode
